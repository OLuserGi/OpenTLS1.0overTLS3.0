# OpenTLS: Legacy Protocol Compatibility Layer

## Описание проекта
Микроприложение для восстановления доступа к старым ресурсам с устаревшими протоколами TLS 1.0.

## Возможности
- Поддержка deprecated TLS-протоколов
- Совместимость со старым оборудованием
- Безопасная обработка legacy-соединений
